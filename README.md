<h1> PortfolioAndroidTest </h1>
<p>
Testing a version of a portfolio website layout as an Android App. This was done using techniques and sources from my CodeFirst Web Development course. 
The design included a responsive navigation bar, and content layout which adjusts to desktop, tablet and mobile screens.

<b>Available to view at: https://emcglackenr1.github.io/PortfolioAndroidTest/ <b>

</p>
